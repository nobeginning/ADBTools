# ADB Library 架构概述

## 项目定位

ADB Library 是一个纯 Kotlin/JVM 实现的 ADB 客户端库，使 Android 应用能够通过 TCP 直连主机上的 ADB Server（端口 5037），使用原生 ADB 协议执行设备管理和调试操作。

## 模块规划

```
ADBTools/
├── adb-lib/                       # 独立 Library 模块 (:adb-lib)
│   └── src/main/kotlin/com/young/sample/adblib/
│       ├── AdbClient.kt           # 公开主入口
│       ├── AdbConfig.kt           # 连接配置
│       ├── AdbException.kt        # 异常定义
│       ├── transport/             # 传输层
│       ├── protocol/              # 协议层
│       ├── service/               # 服务层
│       └── model/                 # 数据模型
├── app/                           # Android 应用模块 (:app)
│   └── src/main/java/.../
│       └── ui/                    # Compose UI 层
└── docs/                          # 设计文档
```

## 数据流

### 单次命令执行

```
App/ViewModel
  → AdbClient.openSession(serial)
    → HostServices.selectDevice(serial)
    → AdbSession.openService("shell:ls")
      → Transport.write(OPEN packet)
      → Transport.read() // OKAY
    → AdbStream.write(WRTE shell command)
    → AdbStream.read() // ShellProtocol stdout/stderr/exit
  → ShellResult
```

### 设备状态追踪（长连接）

```
App/ViewModel
  → AdbClient.trackDevices()
    → HostServices.trackDevices()
      → Transport.write("host:track-devices")
      → Transport.read() // OKAY
      → while(true):
          Transport.read() // 设备列表变化推送
          → emit(parsed devices)
    → Flow<List<AdbDevice>> (持续更新)
```

## 连接模型

```
                    ┌──────────────┐
     App ──────────▶│  AdbClient   │ (一个实例管理一条到 ADB Server 的连接)
                    └──────┬───────┘
                           │ TcpTransport (host:5037)
                    ┌──────▼───────┐
                    │  ADB Server  │
                    └──────┬───────┘
                           │ 内部路由
           ┌───────────────┼───────────────┐
           ▼               ▼               ▼
      AdbSession      AdbSession      AdbSession
      (device-A)      (device-B)      (device-C)
           │               │               │
      AdbStream       AdbStream       AdbStream
      (shell:)        (sync:)         (framebuffer:)
```

- **AdbClient** = 一条到 ADB Server 的 TCP 连接
- **AdbSession** = 通过 `host:transport:<serial>` 选择的一个目标设备
- **AdbStream** = 设备上的一个服务流（shell、sync 等）

## 依赖关系

- **:adb-lib**: 纯 JVM 模块，仅依赖 `kotlinx-coroutines-core` + Java 标准库
- **:app**: Android 模块，依赖 `:adb-lib`
- 零外部第三方依赖
